# Artifact Schema

Required artifacts:

- `00-prd.md`: original or summarized PRD with IDs.
- `01-spec.md`: normalized reviewable Spec.
- `02-spec-review.md`: PRD/Spec review findings and gate result.
- `03-requirement-trace.md`: PRD to Spec to code to tests mapping.
- `04-tech-solution.md`: repo-grounded technical solution.
- `05-solution-review.md`: solution review findings and gate result.
- `06-implementation-tasks.md`: bounded implementation tasks.
- `07-implementation-log.md`: implementation notes and decisions.
- `08-unit-test-plan.md`: unit test design mapped to Spec.
- `09-unit-test-report.md`: command results and coverage gaps.
- `10-delivery-review.md`: final risk review.
- `11-checkpoint.json`: recovery state.
- `12-observability.md`: metrics dashboard.
- `events.jsonl`: append-only event stream.
