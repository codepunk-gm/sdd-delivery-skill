# Team Rules

Customize this file for the team.

Defaults:

- Preserve Chinese text and use UTF-8.
- Follow the repository architecture.
- Keep diffs minimal and reviewable.
- Do not make unrelated formatting-only changes.
- Do not lower test quality to pass checks.
- Spec is mandatory before PRD review.
- Technical solution must be reviewed before implementation.
- Unit test plan and report are required for delivery.
- Keep DevFlow artifacts under `.op/devflow/<feature>/`.
