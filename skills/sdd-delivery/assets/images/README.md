# Image Assets

Place README images here with these recommended names:

- `sdd-delivery-hero.png`
- `workflow-diagram.png`
- `interaction-demo.png`

Recommended sizes:

- Hero image: 1792x1024 or 16:9
- Workflow diagram: 1792x1024 or 16:9
- Interaction demo: 1792x1024 or 16:9

After adding the images, the README image links will render automatically.
